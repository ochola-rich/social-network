# Social Network Project

A full-stack social networking application built with a Next.js frontend and a Go backend.

---

## 🔧 Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd <your-project-folder>

cd frontend
npm install
npm run dev

http://localhost:3000

cd backend
go run backend/cmd/server/main.go

http://localhost:8080
